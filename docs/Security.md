# Security and Health

## Security Check

[https://docs.openclaw.ai/cli/security](https://docs.openclaw.ai/cli/security)

Run regularly:

```bash
openclaw security audit --deep
openclaw security audit --fix
```

## Doctor

[https://docs.openclaw.ai/cli/doctor](https://docs.openclaw.ai/cli/doctor)

Run regularly:

```bash
openclaw doctor
openclaw doctor --repair
openclaw doctor --deep
```

## Update

[https://docs.openclaw.ai/cli/update](https://docs.openclaw.ai/cli/update)

Run regularly:

```bash
openclaw update
openclaw update status
```
