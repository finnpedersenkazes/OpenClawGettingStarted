# Dedicated macOS

A Mac mini or MacBook with Apple Silicon is a good dedicated host, especially if you already use one as a stable always-on machine.

## Advantages

- Quiet and low power consumption
- Reliable hardware
- Good fit for a permanent home lab setup

## When to choose this

- You want a quiet machine running all the time
- You are already comfortable administering macOS
- You do not need NVIDIA GPU acceleration on the host itself

## macOS with NemoClaw

On macOS Apple Silicon, follow NVIDIA's Quickstart for installing Docker on that platform.
