# NVIDIA Brev

NVIDIA Brev is the cloud-oriented path when you want access to a remote NVIDIA GPU environment.

## When to choose this

- You want remote GPU resources
- You prefer a cloud sandbox over a local machine
- Your local machine does not have a suitable NVIDIA GPU
- You want to separate your assistant environment from your everyday computer

## Getting started

Go to <https://www.nvidia.com/en-us/ai/nemoclaw/> and press **Try It Now**.

## How it differs from a local host

- The machine is remote
- The hardware is rented
- The environment is more disposable
- GPU access is the main reason to choose it
