# Dedicated Ubuntu

This is the most straightforward host for both OpenClaw and NemoClaw.

A dedicated Ubuntu installation on a laptop or desktop gives you the fewest moving parts and matches NVIDIA's documented runtime expectations.

## Requirements

| Hardware | Recommended |
| -------- | ----------- |
| CPU      | 4+ cores    |
| RAM      | 16 GB       |
| Disk     | 40 GB       |

| Software | Version                   |
| -------- | ------------------------- |
| OS       | Ubuntu 22.04 LTS or later |
| Node.js  | 22.16 or later            |
| npm      | 10 or later               |

NemoClaw additionally requires **Docker Engine** and **OpenShell**.

## When to choose this

- You want the simplest local setup
- You want to follow NVIDIA's documentation closely (NemoClaw)
- You are comfortable managing a Linux machine
