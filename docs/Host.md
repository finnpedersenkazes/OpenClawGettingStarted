### Prerequisites

| Hardware Requirements | Recommended |
| --------------------- | ----------- |
| CPU                   | 4+ vCPU     |
| RAM                   | 16 GB       |
| Disk                  | 40 GB       |

| Software Dependencies | Version                   |
| --------------------- | ------------------------- |
| Linux                 | Ubuntu 22.04 LTS or later |
| Node.js               | 22.16 or later            |
| npm                   | 10 or later               |
| Docker Desktop        | Installed                 |
| OpenShell             | Installed                 |
