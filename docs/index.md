# OpenClaw Getting Started

Welcome to the OpenClaw Getting Started guide.

## Choose your path

Pick **one** path and follow it straight through. You can explore the others later.

### Path A: OpenClaw on Hostinger

The fastest path. Managed hosting with 1-click deploy -- no server setup needed.

1. [Hostinger VPS](Hosts/Hostinger.md) -- pick a plan and deploy
2. Your agent is live

### Path B: OpenClaw on your own machine

Direct setup on Ubuntu, macOS, or an Ubuntu VM.

1. [Choose a host](Hosts/Ubuntu.md) -- Ubuntu, [macOS](Hosts/MacOS.md), or [Ubuntu VM](Hosts/VirtualUbuntu.md)
2. [AI Models](Models.md) -- pick a provider
3. [Prerequisites](OpenClaw/PreInstallation.md) -- curl, Node.js, Git
4. [Install OpenClaw](OpenClaw/Installation.md)
5. [Onboarding](Onboarding.md)

### Path C: NemoClaw on Ubuntu

NVIDIA's sandboxed workflow with Docker and OpenShell.

1. [Choose a host](Hosts/Ubuntu.md)
2. [Prerequisites](NemoClaw/PreInstallation.md) -- Docker, OpenShell, Node.js
3. [Install NemoClaw](NemoClaw/Installation.md)

### Path D: NemoClaw on NVIDIA Brev

Remote GPU-backed environment.

1. [NVIDIA Brev](Hosts/BrevNVIDIA.md) -- provision the environment
2. [Install NemoClaw](NemoClaw/Installation.md)
